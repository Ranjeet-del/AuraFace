# AuraFace Smart Attendance System

A Face Recognition Based Smart Attendance System using OpenCV and Python.

## Features
- Face Registration
- Face Recognition (LBPH)
- Automatic Attendance Marking
- SQLite Database
- Tkinter GUI

## Tech Stack
- Python
- OpenCV
- SQLite
- Tkinter
